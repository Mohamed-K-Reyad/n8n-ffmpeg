# n8n-ffmpeg

مستودع بسيط لاختبار تكامل n8n مع FFmpeg.

الوصف

هذا المستودع مخصص لتخزين أمثلة وملفات إعداد تتيح تشغيل عمليات تحويل ومعالجة وسائط باستخدام FFmpeg داخل سير عمل n8n.

المحتوى المبدئي

- README.md — الملف الحالي.

كيفية البدء

1. استنساخ المستودع:

   ```bash
   git clone https://github.com/Mohamed-K-Reyad/n8n-ffmpeg.git
   cd n8n-ffmpeg
   ```

2. إضافة ملفات المثال أو سكربتات n8n التي تستخدم FFmpeg.

ملاحظات

- المستودع حالياً فارغ من الملفات إلى جانب هذا README. يمكنك رفع ملفات عبر واجهة الويب أو باستخدام git.
- إذا أردت، أستطيع إضافة مثال n8n workflow أو سكربت استخدام FFmpeg كخطوة تالية.
